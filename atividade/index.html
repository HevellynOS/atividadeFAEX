Atividades 03 e 04

Atv_03: Verificação de número par ou ímpar: Peça ao usuário para digitar um número e informe se ele é par ou ímpar. Fórmula: Use o operador de módulo (%) para verificar se o número é divisível por 2. Se o resultado for 0, é par; caso contrário, é ímpar.

Atv_04: Cálculo de IMC: Solicite ao usuário seu peso (em kg) e altura (em metros) e calcule o Índice de Massa Corporal (IMC), exibindo uma mensagem com a classificação (ex: abaixo do peso, normal, acima do peso).

Atividades 05 e 06

Atv_05: Pergunte ao usuário o valor de um produto e o desconto aplicado (em porcentagem). Calcule o preço com desconto e mostre o resultado.
Fórmula: Preço com desconto = preço original - (preço original * (desconto / 100))

Atv_06: Peça ao usuário para inserir três números e determine qual deles é o maior. Fórmula: Nenhuma fórmula necessária. Compare os números para encontrar o maior.

    Atv_07: Conversor de temperatura: Peça ao usuário para escolher entre converter de Celsius para Fahrenheit ou de Fahrenheit para Celsius e realize a conversão.
    Fórmula (Celsius para Fahrenheit): Fahrenheit = (Celsius * 9/5) + 32
    Fórmula (Fahrenheit para Celsius): Celsius = (Fahrenheit - 32) * 5/9
